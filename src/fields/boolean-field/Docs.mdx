# `booleanField(): ZodField<boolean | undefined>`

A field to represent a 3-state checkbox which will have `true`, `false` or `undefined` value.

</hgroup>

Unlike [checkboxField()](?path=/docs/fields-checkboxfield--docs), the `booleanField`:

- starts with `undefined` value e.g. 'unanswered' state
- when required, it's value can be `false` or `true` for the form to submit (cannot be `undefined`)

```ts
import { booleanField } from "@form-atoms/field";

// required yes/no answer, value must be changed by user to true or false
const yesNoAnswer = booleanField({
  name: "agreesWithFuneralTerms"
  required_error: "Please click yes or no button.",
});
```

Usable with [useCheckboxFieldProps()](?path=/docs/hooks-usecheckboxfieldprops--docs)
