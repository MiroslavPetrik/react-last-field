# `numberField(): ZodField<number | undefined>`

```ts
import { numberField } from "@form-atoms/field";

// required number, must be filled to submit
const heightCentimeters = numberField({
  required_error: "Please set your height.",
});

// optional number, can be empty to submit
const tipAmount = numberField().optional();
```

Usable with [useNumberFieldProps()](?path=/docs/hooks-useNumberFieldProps--docs)
