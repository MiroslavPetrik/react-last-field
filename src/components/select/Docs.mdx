# Select

A combination of generic [useSelectFieldProps](?path=/docs/hooks-useselectfieldprops--docs) and [useSelectOptions](?path=/docs/hooks-useselectoptions--docs)
which renders the options as the children of `<select />`.

```ts
import { Select } from "@form-atoms/field";
```

### Alternative

- [RadioGroup](?path=/docs/components-radiogroup--docs) component is an alternative for a single-choice answer.
- Prefer the `Select` when there are many options to select from.

## Example

```tsx
import {
  Select,
  SelectProps,
  SelectField,
  FieldLabel,
  FieldErrors,
  OptionField,
} from "@form-atoms/field";

const SelectField = <Option, Field extends SelectField>({
  field,
  label,
  ...props
}: {
  label: ReactNode;
} & SelectProps<Option, Field>) => (
  <div style={{ margin: "20px 0" }}>
    <FieldLabel field={field} label={label} />
    <Select {...props} field={field} />
    <FieldErrors field={field} />
  </div>
);
```
